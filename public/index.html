Linxicon bridge solver

Finds a one-word or two-word semantic bridge between two endpoints using the same /api/updateSemantics scoring as linxicon.com. A link counts when similarity is at least 41% (game UI rule). One word in the middle must clear that bar to both endpoints, which is uncommon; the two-hop search is usually where answers appear. After npm run build-graph (long run; npm run build-graph:resume if interrupted), scores live in public/data/ for fast lookups. Open this app at http://localhost:8787 via npm start so /data/ loads. When both endpoints are in that graph, the solver uses only local files (no linxicon.com). Otherwise the server still proxies the API where needed.
